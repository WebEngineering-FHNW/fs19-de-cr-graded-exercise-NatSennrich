Web Enginnering Projekt: Restaurant El Asador - Orders Manager

Restaurant El Asador - Orders Manager ist eine Grails basierte Webapplikation inspiriert vom Restaurant meiner Mutter und hat das Ziel den Bestellungprozess im Betrieb zu vereinfachen, denn aktuell werden die Bestellungen nur auf Papier geschrieben. Was diese Webapplikation anbietet is: 1. Übersicht von verfügbaren Tischen im Restaurant 2. Eine Liste von offenen Bestellung 3. Ein Bereich für die Erfassung neue Bestellungen 4. Übersicht von allen Bestellungen für ein bestimmtes Datum 5. Eine Bestanaufnahme-Seite der Gerichten, sowie der Getränke 6. SpringSecurity Funktionalität
